# flutter-login-signup-sample-page

A beautiful flutter UI with animations

## Platform
* Android
* IOS

## Run
```
git clone https://github.com/Nimatrixx/flutter-login-signup-sample-page.git
cd flutter-login-signup-sample-page
flutter pub get
flutter run
```
